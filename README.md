# CHIP-8

CHIP-8 emulator written in C++ for the purpose of learning about emulation.

![screenshot](docs/screenshot.png)

## Requirements
- C++ compiler (gcc, clang, MSVC)
- CMake
- SDL2

## Compilation

```bash
mkdir build
cd build
cmake -G Ninja ..
ninja
```
